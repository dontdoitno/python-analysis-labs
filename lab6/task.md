Задание 6. Анализ формы распределений и нормализация данных
Цели
Научиться:
•	оценивать форму распределений числовых признаков с помощью асимметрии, эксцесса и коэффициента вариации,
•	принимать обоснованные решения о необходимости нормализации/трансформации,
•	применять подходящие методы нормализации
•	визуализировать исходные и преобразованные распределения с помощью Plotly.
Требования к ноутбуку и задачи EDA выданы в предыдущих заданиях! Они актуальны
 Требования к датасету
1.	Тип:
— Датасет должен быть табличным (структурные данные в виде строк и столбцов).
2.	Количество признаков:
— Должно быть не менее 5 числовых (непрерывных или дискретных) признаков.
— Категориальные признаки могут присутствовать, но не обязательны.
3.	Объём данных:
— Количество строк — от 500 до 100 000 (слишком маленькие датасеты не покажут стабильную форму распределения; слишком большие — неудобны для данной работы).
4.	Природа данных:
— Все числовые признаки должны быть положительными (≥ 0), чтобы можно было корректно применять коэффициент вариации и логарифмические трансформации.
— Не допускаются признаки, у которых среднее значение близко к нулю или отрицательно.
5.	Источник:
— Датасет должен быть взят из открытого источника:
•	Kaggle,
•	UCI ML Repository,
•	Государственные открытые данные (data.gov, data.gov.ru и т.п.),
•	sklearn.datasets, seaborn.load_dataset, statsmodels.datasets и др.
6.	Релевантность:
— Данные должны быть реальными (не синтетическими) и иметь практическую интерпретацию (например, цены, доходы, возраст, количество, площадь, время и т.д.).

 
Задание
Часть 1. Анализ исходных распределений
Для каждого числового признака:
•	Рассчитайте:
•	среднее, медианы, моды (что целесообразно рассматривать?), стандартное отклонение,
•	коэффициент асимметрии (scipy.stats.skew),
•	избыточный эксцесс (scipy.stats.kurtosis(..., fisher=True)),
•	коэффициент вариации.
•	Интерпретируйте:
•	Насколько распределение отличается от нормального?
•	Есть ли признаки с выраженной асимметрией (>1 или <–1)?
•	Какие признаки имеют высокий эксцесс (тяжёлые хвосты)?
•	Какие признаки наиболее/наименее вариабельны?
Часть 2. Визуализация (Plotly)
•	Создайте интерактивные гистограммы, ящики с усами + кривые плотности для всех числовых признаков.
•	Используйте plotly.subplots для компактного отображения.
•	На каждом графике добавьте аннотацию с численными значениями:
Skew: ..., Kurtosis: ..., CV: ...%
Часть 3. Нормализация / трансформация
•	Выберите 2–3 признака с сильной асимметрией или высоким CV.
•	Примените не менее двух методов трансформации, например:
•	Логарифмическая (np.log1p),
•	Квадратный корень,
•	Box-Cox (только для строго положительных данных),
•	Или масштабирование через RobustScaler/PowerTransformer.
•	Для каждого преобразования:
•	Пересчитайте skew, kurtosis, CV.
•	Постройте сравнительные графики «до и после» в Plotly (например, с помощью make_subplots).
Часть 4. Вывод
•	Сформулируйте, какие трансформации оказались наиболее эффективными и почему.
•	Обоснуйте, стоит ли применять эти преобразования перед обучением ML-моделей на этом датасете.
